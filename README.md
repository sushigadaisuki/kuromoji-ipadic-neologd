[kuromoji-js-dictionary](https://github.com/sable-virt/kuromoji-js-dictionary) を使ってビルドしています。  
Nodejsで精度の良い形態素解析がしたいときにどうぞ。  
ファイルが大きいのでGit LFSを使っています。
